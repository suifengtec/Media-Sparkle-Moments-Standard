# Media Sparkle Moments Standard

## segment json

```json

{
    "title": "segment title",
    "desc": "segment description",
    "slug": "for dir path or filename",
    "start_at": "segment start at",
    "endt_at": "segment end at"
}


```

## Media Sparkle Moments 

```json

{
    "title": "media title",
    "desc": "media description",
    "path": "media path",
    "list": [{
            "title": "segment 1",
            "desc": "segment 1 description",
            "slug": "seg_1",
            "start_at": "00:00:05",
            "endt_at": "00:00:15"
        },
        {
            "title": "segment 2",
            "desc": "segment 2 description",
            "slug": "seg_2",
            "start_at": "00:11:05",
            "endt_at": "00:12:15"
        }
    ]
}


```
